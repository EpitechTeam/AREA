# AREA API

API REST du projet AREA

Liens entre plusieurs services:

	- Facebook
	- Instagram
	- Messenger
	- Intra Epitech
	- Outlook
	- Office 365
	- Yammer
	- One Drive

## For development mode :

```sh
$ export NODE_ENV=development

$ npm install

$ npm start

```

## For Production mode :

```sh

$ npm install

$ pm2 start ecosystem.config.js --env production
